# RNN Sentiment Analysis – Deep Learning Web App

RNN Sentiment Analysis is an **end-to-end Deep Learning web application** built using **Python, PyTorch, and Streamlit** to classify movie reviews as **Positive or Negative**.

The project demonstrates **Natural Language Processing (NLP), deep learning model training, text preprocessing, and real-time deployment of an AI model** in a web interface.

---
🔗 Live App: https://github.com/vinod-vaishnav-2005/RNN-sentiment-analysis

## 🚀 Key Features

✅ Interactive **Streamlit web application**
🧠 Deep Learning model using **Recurrent Neural Network (RNN)**
🧹 Text preprocessing using **TF-IDF vectorization**
⚡ Real-time sentiment prediction
📊 IMDB movie review dataset for training
🛡️ Robust inference pipeline for user inputs

---

## 🧠 Machine Learning Workflow

* Text data preprocessing and cleaning
* Tokenization and feature extraction
* TF-IDF vectorization
* Training **RNN model using PyTorch**
* Saving trained model (`.pth`) and vectorizer (`.pkl`)
* Real-time predictions using a **Streamlit UI**
